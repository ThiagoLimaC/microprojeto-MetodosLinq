# Meu Projeto em .NET 8

Este é um projeto de exemplo em .NET 8 que demonstra o uso de várias funcionalidades do LINQ e outras operações em coleções.

## Funcionalidades

- Operações básicas de LINQ (`Where`, `Skip`, `Take`, etc.)
- Operações avançadas de LINQ (`GroupBy`, `Distinct`, `OrderBy`, etc.)
- Manipulação de coleções (`Append`, `Prepend`, `Concat`, etc.)
- Uso de `record` para definir tipos imutáveis

## Instalação

Para instalar e configurar o projeto, siga os passos abaixo:

1. Clone o repositório:

`git clone https://github.com/seu-usuario/seu-repositorio.git`

2. Navegue até o diretório do projeto:

`cd seu-repositorio`

3. Restaure as dependências:

`dotnet restore`
